# Elyas Heidari - Research Engineer

Professional portfolio and research blog of Elyas Heidari, Research Engineer in Computational Biology.

## Tech Stack
- **Framework**: Next.js (App Router)
- **UI Library**: Material UI (v6) - Dark Theme with Google Design aesthetics.
- **Deployment**: GitHub Pages (Static Export).

## Features
- **Portfolio**: Showcasing projects like `segger` and `SageNet`.
- **Blog**: Technical writings on Spatial Genomics and Machine Learning.
- **Reading List**: A curated list of influential books and papers.
